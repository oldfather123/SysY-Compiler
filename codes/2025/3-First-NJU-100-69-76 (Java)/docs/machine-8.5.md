# 机器指令生成

将LIR翻译为汇编

## 稀疏变量初始化

对于零占比到达一定比例的栈上变量，会自动使用memset填充0后再填写非0部分

## Spilling

对于在寄存器分配时放在栈上的变量，整数和浮点各保留了三个寄存器用于进行栈上读写的缓存

缓存使用CLOCK机制选择每次淘汰的变量，同时在每个函数调用之前和基本块结束时强制写回

## 寄存器保存

对于caller-saved寄存器，调用之前进行保存，复原则按需进行，以优化连续进行函数调用时的性能

## 并行移动翻译

对于并行移动，移动的顺序构成一外向基环树森林

每次选择没有出边的位置进行移动，有环则选其中一个位置进行备份，将环断开后继续
