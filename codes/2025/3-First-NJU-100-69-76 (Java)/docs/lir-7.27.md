# LIR

## RC64GC

仍然为SSA形式，Phi转换为基本块末尾的RVParallelMove

指令: 包括RV系列指令和br、call、callvoid、ret、retvoid

需要实现的基本函数: void _rv64gc_memzero(i64 pointer, i64 size), 将pointer指向的长度为size的内存空间全部置为0

### 分析

实现了LIR版本的CFG和支配分析

还有变量的存活区间分析，用于辅助寄存器分配

以及在寄存器分配完后有每个函数的寄存器使用情况分析，用于辅助机器代码生成

### 优化

仍然有控制流化简、常量折叠、公共表达式消除、指令合并和强度削减的简化版本

并进行常量和全局变量地址的显式化（使用li和la加载）

### 寄存器分配

在寄存器分配之前先进行块内指令调度，在考虑寄存器压力的同时尽量将无依赖关系的指令放在一起，加速程序执行

基于从IR的频率分析继承的结果，对变量spill和调用前临时保存的代价进行估计，在SSA上进行图染色后分配寄存器

寄存器分配完成后，消除对同一位置的并行移动，并再进行一次控制流图化简
